Monero Pool UI

Welcome to Monero Pool UI

A monitoring portal for all your Monero mining pools.

Features

  • Register your own pool.
  • Browse all registered pools.
  • Find your payment addresses across all registered pools.
  • Monitor your payment address.

Technology

  • Built with Angular framework and Angular Material.
  • Works with any nodejs-pool server.
  • Open source project.
  • Active development team.
